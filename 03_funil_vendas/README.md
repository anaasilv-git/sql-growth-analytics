# Funil de Vendas / Funil de CRM

Aqui estão consultas SQL voltadas para construir e analisar funis de conversão.

Nesta pasta você encontra queries para:
- calcular cada etapa do funil
- medir taxa de conversão entre etapas
- identificar gargalos
- analisar comportamento do cliente ao longo da jornada

Essas métricas são essenciais para estratégias de Growth e otimização de performance.
