# Base de Dados

Esta pasta armazena a documentação das tabelas utilizadas nos exercícios de SQL.

Aqui organizo:
- dicionário de dados
- estrutura das tabelas
- relações entre tabelas
- exemplos de bases fictícias usadas para simulação

O objetivo é manter uma visão clara da modelagem para facilitar análises de CRM, funis, cohorts e métricas de retenção.
