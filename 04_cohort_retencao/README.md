# Cohorts e Retenção

Esta pasta contém análises SQL focadas em retenção de clientes ao longo do tempo.

Inclui:
- criação de cohorts mensais
- cálculo de retenção por período
- análises de comportamento longitudinal
- identificação de padrões de churn e engajamento

Cohorts ajudam a entender se as ações de CRM estão funcionando e como a base evolui mês a mês.
