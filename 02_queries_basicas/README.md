# Queries Básicas de SQL

Nesta pasta ficam consultas SQL simples e diretas, que resolvem problemas comuns em cenários de CRM e Growth.

Inclui:
- contagem de clientes ativos
- pedidos nos últimos 30 dias
- primeiras métricas de engajamento
- filtros básicos e agregações

O foco é construir uma base sólida para análises mais complexas como funis e cohorts.
