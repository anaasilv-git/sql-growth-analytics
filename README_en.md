# SQL Growth Analytics

This repository is dedicated to studying, simulating, and practicing SQL applied to **CRM, growth, customer behavior, and data-driven decision-making**.

Here, I organize real and structured SQL queries that support analyses such as:
- identifying and engaging active customers  
- building and measuring conversion funnels  
- cohort analysis and longitudinal customer behavior  
- churn, frequency, and value metrics  
- essential KPIs for CRM and Growth strategies  

This project reflects my approach as a data-driven economist:  
**understanding patterns, optimizing resources, predicting behavior, and transforming information into action.**
